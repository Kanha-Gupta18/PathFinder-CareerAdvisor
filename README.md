# PathFinder - Personalized College Advisor

## Features
- AI-Powered Quiz: Uses Google Gemini to generate custom aptitude questions.
- College Prediction: Predicts admission chances based on JEE, NEET, and CUET scores.
- **Career Roadmap: Generates a personalized career path using AI.

## How to Use
1. Open the [Live Demo Link Here].
2. Enter your Gemini API Key when prompted (required for AI features).
3. Select your stream and take the quiz!

## Tech Stack
- HTML5
- Tailwind CSS
- Vanilla JavaScript
- Google Gemini API
